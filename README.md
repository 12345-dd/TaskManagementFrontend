# Job Analytics — Frontend

A full-stack application that helps to organize the tasks by adding, updating and deleting tasks.

**Live Demo:** [https://taskifyflow.netlify.app](https://taskifyflow.netlify.app)  
**Backend Repo:** [https://github.com/12345-dd/TaskManagementBackend](https://github.com/12345-dd/TaskManagementBackend)

---

## Features

- **JWT-based Authentication** — Secure signup & login
- **Task Management** — Add, update, and manage Tasks
- **Protected Routes** — Only authenticated users can access their data

---

## Getting Started (Local Setup)

### 1. Clone the Repository
```bash
git clone https://github.com/12345-dd/TaskManagementFrontend
cd TaskManagementFrontend
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the App
```bash
npm run dev
```

> App runs on `http://localhost:5173`
